System.Portable
A library supporting composable applications wherever .Net runs.
.Net 4.5, SL5, Windows (of all shapes and sizes), Xamarin.Touch

Current: Gamma (pre-alpha) 


Release Highlights:
0.1.* (Gamma):
	* Partial API - Flushed out from a major rewrite.
	* Providers for IOC, Types, Reflection and more.
	* Extension methods providing all manner of conveniences
	* Support for Autofac, Caliburn Micro, and OWIN.
	* Support comming for MVC, Web Api 2, Signalr (Yet to be updated to new API)

THIS CODE IS EXPERIMENTAL AND THE CODEBASE IS IN CONSTANT FLUX.
